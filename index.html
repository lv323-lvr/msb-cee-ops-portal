Internal SOP

         

Custom Executive Education Operations Portal

         

            A lightweight, searchable home for planning, delivery, and closeout workflows.          

       
     
       
       

Overview

       

          This site is designed to centralize operational guidance for MSB Custom Executive Education           programs. Use it as a practical reference point for program planning, live delivery, and closeout.        

         
         
           

Primary Goals

           
                 
  • Reduce scattered SOP documentation
    •              
  • Standardize workflows across programs
    •              
  • Support continuity and onboarding
    •              
  • Make execution faster and clearer
    •            
         
         
           

Recommended Use

           
                 
  • Reference during program planning
    •              
  • Checklists before live sessions
    •              
  • Post-program reconciliation
    •              
  • Training new team members
    •            
         
       
     
       
       

Phase 1: Planning

       

          Use this phase to align timelines, stakeholders, scope, and required deliverables.        

         
          1. Intake and Scope Confirmation          
           
                 
  • Confirm client objectives and target audience
    •              
  • Review delivery format: in-person, virtual, hybrid
    •              
  • Clarify program dates, session times, and faculty needs
    •              
  • Identify dependencies, risks, and special requirements
    •            
         
       
         
          2. Stakeholder Mapping          
           
                 
  • Identify internal program owner
    •              
  • Identify faculty and content contributors
    •              
  • Identify client-side points of contact
    •              
  • Define decision-makers and approvers
    •            
         
       
         
          3. Timeline and Milestones          
           
                 
  • Establish major planning checkpoints
    •              
  • Set content deadlines
    •              
  • Set registration and communication milestones
    •              
  • Schedule rehearsal / tech check if applicable
    •            
         
       
         
          4. Planning Outputs          
           
                 
  • Program timeline
    •              
  • Responsibility matrix
    •              
  • Session structure outline
    •              
  • Known risks and open questions tracker
    •            
         
       
     
       
       

Phase 2: Build

       

          Prepare content, logistics, communications, and platform setup.        

         
          1. Content Assembly          
           
                 
  • Collect slide decks and materials
    •              
  • Review for branding, formatting, and accessibility
    •              
  • Confirm version control process
    •            
         
       
         
          2. Platform and Logistics Setup          
           
                 
  • Create Zoom / virtual session infrastructure
    •              
  • Confirm room logistics for in-person elements
    •              
  • Coordinate off-site meals, transportation, and signage as needed
    •            
         
       
     
       
       

Phase 3: Delivery

       

          Focus on smooth execution, participant experience, and rapid troubleshooting.        

         
          1. Day-of Execution          
           
                 
  • Confirm faculty arrival / login status
    •              
  • Monitor schedule pacing
    •              
  • Track attendance and participant needs
    •              
  • Coordinate transitions, breaks, and lunch timing
    •            
         
       
         
          2. Live Support          
           
                 
  • Manage chat and tech issues
    •              
  • Support faculty and moderators
    •              
  • Escalate issues early
    •            
         
       
     
       
       

Phase 4: Closeout

       

          Wrap the program cleanly and preserve what will help the next one run better.        

         
          1. Post-Program Actions          
           
                 
  • Collect evaluations and feedback
    •              
  • Archive final materials
    •              
  • Document lessons learned
    •              
  • Update SOPs based on real-world experience
    •
